#  :notebook:Spring boot - Todolist Project 

Spring boot & Vue 기반 투두리스트 프로젝트

## :warning:Requirements

* Open JDK 17
* Gradle v8.5
* Spring boot v3.2.3
* use lombok lib
* use JPA
* use `mariadb` database 

### :computer:install
```bash
$ git clone git@github.com:seilyn/todolist.git

or

$ git clone https://github.com/seilyn/todolist.git
```

### :fire:run
run with `gradle`

```bash
$ gradlew build 
```
