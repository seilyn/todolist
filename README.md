# Spring boot - Todolist Project 

투두리스트 프로젝트

## Requirements

* Open JDK 17
* Gradle v8.5
* Spring boot v3.2.3
* use lombok lib
* use JPA
* use `mariadb` database 

### install
```bash
$ git clone git@github.com:seilyn/todolist.git

or

$ git clone https://github.com/seilyn/todolist.git
```

### run
run with `gradle`

```bash
$ gradlew build 
```
